# 小维的导航站

一个为开发者打造的 **个人技术导航站**，采用赛博朋克风格 UI，支持 **自适应布局（vw/vh/rem）**、**三行固定排版**、**流光卡片**、**粒子背景**、**能量流动画**，并为每个站点提供简洁的一行描述。

本项目用于整合个人常用站点、工具、知识库与后台系统，让所有入口集中在一个页面中，快速访问、高效管理。

## 项目结构

```
/
├── index.html        # 主页面
├── avatar.jpg        # 头像
├── favicon.ico       # 网站图标
└── /assets           # 可选：图片、脚本、样式等
```

## 技术栈

- **HTML5**
- **CSS3（vw/vh/rem 自适应）**
- **无框架纯前端实现**
- **无依赖、无构建、开箱即用**



## 功能特点

### 赛博朋克 UI

- 粒子星空背景
- 左右能量流光动画
- 中央光晕氛围效果
- 玻璃拟态卡片 + 霓虹流光边框
- 悬浮缩放 + 光效增强

### 完全自适应布局

- 基于 **1920×1080** 视觉比例设计
- 使用 **vw / vh / rem** 实现全局自适应
- 缩放（125%、150%）不变形
- 不出现滚动条或布局错乱

### 稳定的三行排版

- 固定 3 行
- 自动扩展列数
- 卡片数量可增减

### 卡片描述支持

- 每个站点卡片支持一行描述
- 超出自动省略号
- 不改变卡片外观与高度
